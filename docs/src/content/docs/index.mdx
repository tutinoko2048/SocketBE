## What is SocketBE?

SocketBE is a powerful Node.js library that allows you to interact with Minecraft Bedrock Edition through WebSockets. Build custom applications, automation tools, bots, and more with full TypeScript support.

## Key Features

<CardGrid stagger>
	<Card title="Typed Event Handling" icon="seti:typescript">

	<Card title="Command Execution" icon="seti:powershell">

	</Card>
